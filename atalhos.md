# ⌨️ Guia Básico de Atalhos do Windows

Este repositório contém atalhos essenciais do Windows para aumentar produtividade no uso diário.

---

## 🪟 Sistema e Navegação

| Atalho | Descrição |
|--------|----------|
| `Win` | Abre o menu iniciar |
| `Win + D` | Mostra/oculta a área de trabalho |
| `Win + E` | Abre o explorador de arquivos |
| `Win + L` | Bloqueia o computador |
| `Win + I` | Abre configurações |
| `Win + A` | Abre central de ações |
| `Win + S` | Abre busca |

---

## 🗂️ Gerenciamento de Janelas

| Atalho | Descrição |
|--------|----------|
| `Alt + Tab` | Alterna entre janelas |
| `Win + Tab` | Visão de tarefas |
| `Win + ← / →` | Ajusta janela para lados |
| `Win + ↑ / ↓` | Maximiza/minimiza |
| `Win + Shift + ← / →` | Move janela entre monitores |

---

## ✂️ Edição e Produtividade

| Atalho | Descrição |
|--------|----------|
| `Ctrl + C` | Copiar |
| `Ctrl + V` | Colar |
| `Ctrl + X` | Recortar |
| `Ctrl + Z` | Desfazer |
| `Ctrl + Y` | Refazer |
| `Ctrl + A` | Selecionar tudo |
| `Ctrl + S` | Salvar |

---

## 📋 Área de Transferência

| Atalho | Descrição |
|--------|----------|
| `Win + V` | Histórico da área de transferência |
| `Ctrl + Shift + V` | Colar sem formatação (em alguns apps) |

---

## 🖥️ Múltiplas Áreas de Trabalho

| Atalho | Descrição |
|--------|----------|
| `Win + Ctrl + D` | Nova área de trabalho |
| `Win + Ctrl + ← / →` | Alterna entre áreas |
| `Win + Ctrl + F4` | Fecha área atual |

---

## 🔧 Ferramentas do Sistema

| Atalho | Descrição |
|--------|----------|
| `Ctrl + Shift + Esc` | Gerenciador de tarefas |
| `Win + R` | Executar |
| `Win + X` | Menu avançado |
| `Win + Pause` | Informações do sistema |

---

## 🧠 Dicas

- Pare de usar mouse para tudo — isso é perda de tempo cumulativa
- Decore os atalhos que você usa todo dia, não todos de uma vez
- Use múltiplas áreas de trabalho se você se distrai fácil
